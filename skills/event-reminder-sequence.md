# Skill: Event Reminder Sequence

## Purpose
Creates timed reminder sequences for registered event attendees to maximize show-up rates and build excitement leading up to the event.

## When to Use
- After someone registers for an event
- Building pre-event communication cadence
- Creating countdown communications

## Inputs Required
- **Event name and branding**: (Modern Medicine Symposium / Art of the Eyes)
- **Event dates and location**
- **Reminder cadence**: How many reminders and when (e.g., 2 weeks, 1 week, 3 days, day before, day of)
- **Agenda/schedule** (if available): To tease specific sessions
- **Logistics info**: Parking, check-in time, dress code, what to bring

## Standard Reminder Cadence
```
Reminder 1: 2 weeks before — Build excitement, share agenda highlights
Reminder 2: 1 week before — Logistics + what to expect
Reminder 3: 3 days before — Final prep + tease a highlight
Reminder 4: Day before — Quick "see you tomorrow" + logistics recap
Reminder 5: Day of (morning) — Welcome + check-in details
```

## Output Format
For each reminder:
```
---
Reminder [#] | Send: [X days/hours before event]
Subject: [subject line]
---
[Body]
---
```

## Style Rules
- Each reminder should have a DIFFERENT angle (don't just repeat "don't forget!")
- Build momentum — excitement should increase as the event approaches
- Include practical logistics in at least 2 reminders
- Day-of email should be short and energetic
- NO em dashes
- Maintain the warm, collegial BTL tone

## Quality Checks
- [ ] Each reminder has a unique angle
- [ ] Logistics covered at least twice
- [ ] Correct event branding
- [ ] No em dashes
- [ ] Timing makes sense relative to event date
- [ ] Day-of email is concise
