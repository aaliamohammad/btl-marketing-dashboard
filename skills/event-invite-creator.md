# Skill: Event Invite Creator

## Purpose
Creates compelling event invitation packages for BTL symposium events, including the initial invite email, supporting materials copy, and registration CTAs.

## When to Use
- Launching a new event campaign
- Creating invitations for a specific practice segment
- Building the initial touchpoint for event outreach

## Inputs Required
- **Event name**: "The Modern Medicine Symposium" or "The Art of the Eyes" (eye doctors only)
- **Event dates and location**: e.g., April 24-26, Hyatt Regency Tulsa Downtown
- **Target segment**: Practice type(s) being invited
- **Key speakers or agenda highlights** (if available)
- **Registration method**: Link, reply, or phone
- **Urgency factors**: Limited spots, early bird pricing, deadlines

## Output Format
```
---
EVENT INVITATION PACKAGE
Event: [name]
Dates: [dates]
Location: [venue]
Target: [practice types]
---

## Primary Invite Email
Subject: [subject line]
[Body]

## Short-Form Invite (for SMS or social)
[2-3 sentence version]

## Key Talking Points (for phone outreach)
- [Point 1]
- [Point 2]
- [Point 3]

## Registration CTA Options
- [Option 1: email reply]
- [Option 2: link]
- [Option 3: phone]
```

## Branding Rules
- **Eye doctor practices**: Use "The Art of the Eyes" branding throughout
- **All other practice types**: Use "The Modern Medicine Symposium" branding
- Never mix branding in a single communication
- Always mention both upcoming events when there are multiple (e.g., Tulsa in April + Dallas in June)

## Style Rules
- Lead with what they'll gain (CE credits, hands-on experience, networking)
- Make it feel exclusive, not mass-marketed
- Include logistical details but don't overwhelm
- Warm, inviting tone — like a colleague sharing an opportunity
- NO em dashes
- Short paragraphs

## Quality Checks
- [ ] Correct event branding for target practice type
- [ ] Dates, location, and venue are accurate
- [ ] Registration method is clear
- [ ] No em dashes
- [ ] Both events mentioned if applicable
- [ ] Tone is inviting, not salesy
